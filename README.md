# Finance-Viewer-2025-V1-HJSP

A Finance Viewer allowing people to visualise their finances overtime. It includes options for linear, and compounding growth to view how debt, investments, sources of income, and other various means of money flow may affect over networth.

The application runs entirely in the browser using HTML, JavaScript, and Chart.js.

## Features

* Visualize wealth growth over time
* Add multiple income or investment flows
* Choose between:

  * Linear increases
  * Percentage-based (compounding) increases
* Adjustable time horizon
* Individual or combined graph display
* Dynamic chart updates

## How It Works

Users can define different "money flows", each with:

* A name
* An increase amount
* A frequency (how often the increase occurs)
* A mode:

  * Linear
  * Percentage (compounding)
* Optional initial value for percentage calculations

The application then calculates total wealth across the selected time period and displays the results using a line chart.

## Technologies Used

* HTML
* CSS
* JavaScript
* Bootstrap
* Chart.js

## Usage

1. Open the website.
2. Set the time period.
3. Enter an initial value if desired.
4. Add money flows using the **Add Money Flow** button.
5. Adjust parameters and watch the chart update automatically.

## Deployment

This project is hosted using GitHub Pages and runs entirely as a static website.
